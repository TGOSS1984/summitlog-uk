# SummitLog UK

A full-stack UK mountain completion tracker for logging progress across Wainwrights, Munros, Welsh Nuttalls, and other major UK mountain collections.

Users will be able to create an account, browse a master list of mountains, mark summits as completed or planned, add route notes, completion dates, hike distance, personal descriptions, and uploaded images.

The project will include interactive maps, progress dashboards, region-based completion stats, filtering, editing, and a premium mountain-inspired UI using scalable CSS design tokens.
